# carte-ACIA_RS232

Cette platine se branche sur la platine d'extension de bus et permet d'ajouter une liaison série RS232 à l'ORIC (de 50 à 19200 bauds). 
Elle occupe les mêmes adresses que l'ACIA du telestrat soit 31C-31F
